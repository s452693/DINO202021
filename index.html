git branch -d "branch"

Komenda branch operuje na gałęziach, może je przykładowo tworzyć lub listować. Parametr -d usuwa daną gałąź "branch", przy czym można podać ich więcej niż jedną. Jeśli jakaś gałąź posiada reflog, to również zostanie on usunięty. Gałąź musi być w pełni zmergowana (scalona) z gałęzią powyżej (upstream) albo HEAD. /p>
